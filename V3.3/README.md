# 多层嵌套压缩解压工具

## 系统要求
支持 Windows 10 / Windows 11，建议 64 位系统。

## 运行方式
本工具为单文件绿色版，无需安装 Python。

双击“多层嵌套压缩解压工具.exe”即可运行。

## 必要依赖
工具优先使用本机已安装的 7z；本机没有时，使用单文件程序内置的 7z 兜底。

MP4 内嵌 ZIP 使用 ZIP 段提取方案；工具会提取 ZIP 有效段到本次临时目录，再使用 Python ZIP 解压。

## 绿色运行说明
本工具按绿色软件策略运行：

程序关闭后，只保留用户选择的输出目录中的解压结果。

运行过程中产生的临时目录、运行时缓存、诊断缓存和失败材料会自动清理。

MP4 内嵌 ZIP 解压失败时，也不会保留临时 ZIP。

## 输出目录
支持本地磁盘、移动硬盘、映射盘和局域网共享路径。

输出目录为空时，默认输出到源文件所在文件夹。
