# 多层嵌套压缩解压工具 使用说明

## 支持系统
Windows 10 / Windows 11 64 位。

## 支持格式
ZIP、7Z、TAR、GZ、TGZ、BZ2、XZ、无扩展名、错误扩展名和常见分卷。

## 最大层数
最多自动处理 11 层。

## 密码池
多个密码用中文或英文逗号分隔。

示例：
孙笑川258,123456,abc

密码池不会自动排序。

## MP4 内嵌 ZIP
部分特殊 MP4 文件开头是正常 MP4，但文件中间或尾部包含 ZIP 数据。

当密码池包含“孙笑川”时，工具会优先检查 MP4 内嵌 ZIP。

工具会从检测到的 ZIP 起始位置提取有效 ZIP 段到缓存目录，再使用 Python ZIP 解压。

原 MP4 不会被改名、删除或裁切。

## 7z
工具内置 7z.exe / 7z.dll，用于处理 7Z、TAR、GZ、TGZ、BZ2、XZ 和分卷。

## 输出目录
输出目录为空时，默认使用源文件所在目录。
