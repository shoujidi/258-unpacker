tools\7z\License.txt

## 绿色运行
发布产物只包含单个 exe：

多层嵌套压缩解压工具.exe

程序优先使用本机已安装的 7z；本机没有时，使用单文件程序内置的 7z 兜底。

程序运行时可能短暂产生 .runtime_cache、cache、._nested_unpacker_work 等目录；正常关闭、停止任务或异常退出时会自动清理。

关闭后只保留用户输出目录中的解压结果，不保留诊断缓存、失败材料或临时 ZIP。

## MP4 内嵌 ZIP
MP4 内嵌 ZIP 使用 ZIP 段提取方案：

- 保留 detector 判断逻辑
- 使用 embedded_zip_offset
- 从 offset 开始提取 ZIP 有效段到本次临时目录
- 使用 Python ZIP 解压
- 成功或失败后都删除临时 ZIP
