# 多层嵌套压缩通用解压工具
解放双手
【系统要求】
支持 Windows 10 / Windows 11，建议 64 位系统。

【运行方式】
本工具为单文件绿色版，无需安装 Python。
双击“孙笑川258专用解压工具.exe”即可运行。

【必要依赖】
本机需要安装 WinRAR。
建议路径：C:\Program Files\WinRAR\WinRAR.exe

【缓存说明】
首次运行时，程序目录下可能出现隐藏文件夹：
.runtime_cache：单文件 exe 运行时缓存
cache：解压任务中间缓存

正常关闭程序后，运行时缓存会自动清理。
如果程序被强制关闭，缓存文件夹可能残留，可手动删除。

【输出目录】
支持本地磁盘、移动硬盘、映射盘和局域网共享路径。
输出目录为空时，默认输出到源文件所在文件夹。
