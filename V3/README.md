# 孙笑川258专用解压工具

## 系统要求
支持 Windows 10 / Windows 11，建议 64 位系统。

## 运行方式
本工具为单文件绿色版，无需安装 Python。

双击“孙笑川258专用解压工具.exe”即可运行。

## 必要依赖
工具内置 7z，用于处理 7Z、TAR、GZ、TGZ、BZ2、XZ 和分卷。

MP4 内嵌 ZIP 使用 ZIP 段提取方案；工具会提取 ZIP 有效段到缓存目录，再使用 Python ZIP 解压。

## 缓存说明
首次运行时，程序目录下可能出现缓存文件夹：

.runtime_cache：单文件 exe 运行时缓存

cache：解压任务中间缓存

正常关闭程序后，运行时缓存会自动清理。若 MP4 内嵌 ZIP 解压失败，相关临时 ZIP 可能保留在 cache 中便于人工检查。

## 输出目录
支持本地磁盘、移动硬盘、映射盘和局域网共享路径。

输出目录为空时，默认输出到源文件所在文件夹。
